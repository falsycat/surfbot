# Card Format Instructions

Generate concise, informative Kanboard cards for news and article items.

Set **only** the fields listed below. Do not set any other fields unless explicitly instructed elsewhere.
If other instructions (e.g. feed-specific instructions) conflict with these defaults, follow those instructions.

- title: Clear, descriptive title (keep original language if specified by feed instruction)
- description: |
    Write a self-contained summary in Markdown that lets the reader understand the key points
    **without reading the original article**. Include:
    - What the article is about (1-2 sentences of context)
    - The main findings, decisions, or techniques covered (bullet points preferred)
    - Any notable numbers, names, or conclusions worth remembering
    - The source URL as a link at the end

    Aim for 150-300 words. Omit filler phrases. Prefer concrete facts over vague impressions.
